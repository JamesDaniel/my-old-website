Useful Git Code

The following code assumes you have access to a server with git installed on it. Or just create a github account and use that.


Setup Git

git config --global user.name "James McGarr" Set name to be associated with your commits.

git config --global user.email jamesmcgarr@example.com Set your email.

git config --global color.ui true Add colours to your command line.

Working With A Local Repository

git init Initialize a local git repository.

git commit -m "comment goes here" Committing changes to repository.

Working With A Remote Repository

git remote add origin https://github.com/your-name/remote-repo.git Link to a remote repository.

git push -u origin master push local repo to remote repo.
The -u modifier saves origin master so next time just type git push

git pull origin master Pull down any new changes to remote repo

git clone ⟨URL⟩ Get a copy of an existing Git repository.

Working With The Staging Area

git add ⟨file⟩ Add file to staging area.

git add --all Add all new, modified or deleted files to staging area.

git add '*.txt' Use wildcard to add all changes to staging area.

git reset ⟨file⟩ Unstage file.

View Status, Log And Other Information

git status Check status of git repository.

git log Check the history of changes.

git diff HEAD Look at what is different from the last commit
The HEAD part is used to refer to the most recent commit.

git diff --staged See changes that were just made using git add.
--staged refers to what has been added but not committed.

Working With Branches

git branch new_branch Create a separate copy of your repository (aka. Branch) called "new_branch" or whatever you want to name it.

git branch See a list of local branches.

git checkout new_branch Switch to the "new_branch" branch.

git merge new_branch Assuming you have switched back to the master branch, use this command to merge the "new_branch" to the "master" branch.

git branch -d new_branch Delete the "new_branch" branch.

Deleting And Undo

git checkout -- ⟨file⟩ Get rid of all the changes since the last commit for ⟨file⟩

git rm '*.txt' Remove .txt files from repository and stage the removal so when you run the command git commit it will complete the removal from the repository.
